🏠  КАТАЛОГ ОБЪЯВЛЕНИЙ

─────────────────────────────────────────────────────────────────────────────

• Управление верификацией → только администраторы
• Модерация контента → только администраторы
• Доступ к чужим объявлениям → только владелец или администратор
• Просмотр чужих личных данных → только свой профиль или администратор

🔄  ТИПИЧНЫЙ СЦЕНАРИЙ РАБОТЫ:
─────────────────────────────────────────────────────────────────────────────

1.  ПОИСК ЖИЛЬЯ:
    Гость → Главная страница → Поиск/фильтры → Просмотр объявлений
    
2.  КОНТАКТ С ВЛАДЕЛЬЦЕМ:
    Пользователь → Регистрация → Вход → Поиск → Написать сообщение
    
3.  РАЗМЕЩЕНИЕ ОБЪЯВЛЕНИЯ:
    Владелец → Личный кабинет → Добавить объявление → Заполнение формы → Публикация
    
4.  ПОДАЧА ЗАЯВКИ:
    Арендатор → Поиск → Выбор жилья → Заявка → Ожидание ответа
    
5.  ВЕРИФИКАЦИЯ:
    Пользователь → Профиль → Запрос верификации → Загрузка документов → Ожидание
    
6.  МОДЕРАЦИЯ:
    Администратор → Панель админа → Проверка объявлений → Одобрение/Отклонение
